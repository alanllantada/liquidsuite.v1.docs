# Error Response

Liquid Suite uses conventional HTTP response codes to indicate the success or failure of an API request. In general: Codes in the 2xx range indicate success. Codes in the 4xx range indicate an error that failed given the information provided (e.g., a required parameter was omitted, a charge failed, etc.). Codes in the 5xx range indicate an error in the server.

## HTTP STATUS CODE

200 - OK
201 - CREATED
400 - BAD REQUEST
401 - UNAUTHORIZED
